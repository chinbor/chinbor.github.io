vuepress + vuepress-theme-reco 搭建个人站点

chinbor2022-09-21 08:00:00frontEndvuepress vuepress-theme-reco

# 需求分析

作为一个正在工作的前端儿，我的需求就是：

高颜值、快速、简单且高效

# 方案选择

  1. antfu的私人站点 antfu.meopen in new window
  2. hexoopen in new window
  3. 从0-1构建一个自己的站点（需求分析 -> ui设计 -> 具体编码）
  4. vuepressopen in new window

纵观以上方案，可以很简单的得出结论：

请选择 vuepress，真香~~~

但是！！！他的默认主题真的很丑，于是一波调研后选择了 vuepress-theme-recoopen in new window 为配置主题

# 具体操作

vuepress-theme-reco 的官网已经描述的很详细了，实在不行可以去仿造作者的个人站点进行配置（我就是这样干的...）
